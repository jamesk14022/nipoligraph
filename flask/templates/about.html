About

Data are updated roughly every week, using sources:

  • Assembly details from the official NI Assembly site; in particular, the AIMS Portal
  • Tweets from Twitter, gathered using rtweet
  • News articles from NewsRiver
  • Polling data from any relevant source, which for Northern Ireland is almost exclusively LucidTalk

Plots are created with Altair, a Python interface to Vega-Lite. Other displays use a few JavaScript/jQuery libraries, including DataTables, Sparklines, and ProgressPieSVG. The data science parts of the site use a little , and more Python: NumPy, pandas, Gensim, NLTK, scikit-learn, and SOMETHING FOR GPT-2?

The site is built with Python and Flask and is hosted on .
